# 사용후기

**Canva**는 가볍게 이용할 수 있는 편집도구, AI 음성 서비스, AI 영상 제작 서비스를 동시에 제공한다.      
게다가 Web 기반 편집 도구가 있어 굉장히 편하게 사용 가능하다.    
그리고 현재 무료로 사용 가능하다. 

기타 여러 도구를 사용하는 것 보다, Canva 하나로 제작하는게 퀄리티가 더 좋은 것 같다!
